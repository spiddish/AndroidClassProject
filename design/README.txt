User Stories
1.  User can view his/her current bookings
2.  User can delete current booking
3.  User can book a room by placing a pin on the map indicating their location
4.  User can filter bookings based on the following options:
    - number of people
    - time
    - date
    - length of time needed
    - projector in room
    - polycom in room
5.  User can view all rooms that match their filter criteria then book by touching one of the listings
